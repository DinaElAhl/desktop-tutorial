# E² Teaching Framework

> **Built with Claude (Anthropic AI)** — April 10, 2026
>
> A complete teaching companion app built around the core philosophy **"Every Student Has the Right to Understand"**. This project was created collaboratively with Claude AI to produce six fully-functional, standalone HTML/JSX deliverables for educators.
>
> ---
>
> ## 🎯 Philosophy
>
> > *It is never the student's problem — the teacher adapts first.*
> >
> > The E² (Engage & Empower) Teaching Framework is built on a warm terracotta + dusty rose + soft sand palette and centers on four pillars of effective teaching, culminating in the **Student's Bill of Learning Rights** manifesto.
> >
> > ---
> >
> > ## 📁 Files
> >
> > | File | Description |
> > |------|-------------|
> > | `e2-teaching-framework.html` | Printable reference guide with all 4 pillars, Student's Bill of Learning Rights manifesto, and interactive checklist |
> > | `e2-complete-app.html` | All-in-one tabbed web app with lesson planner, blueprint/LMS planner, framework reference, checklist, and export |
> > | `e2-teaching-framework-interactive.jsx` | React component version with tabs, lesson planner, blueprint grid, framework accordion, and progress tracking |
> > | `e2-lesson-planner.html` | Step-by-step guided lesson plan builder with all 4 pillars, 5-part blueprint, differentiation, engagement levers |
> > | `e2-blueprint-planner.html` | Weekly spreadsheet-style planner with Week, Theme, Lesson Objectives, Lesson Links, Nearpod, Flip, Classwork, Homework, Assessment, Teachers Guide, Comments |
> > | `e2-rubrics-assessment.html` | Engagement rubric, self-reflection checklist, quick-check generator, and classroom observation walkthrough form |
> >
> > ---
> >
